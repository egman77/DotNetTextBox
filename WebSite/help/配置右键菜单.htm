DotNetTextBox控件配置个性化右键菜单功能

一、配置向导

配置控件右键菜单功能属性:RightClickMenuConfig

配置规则:功能名1,功能名2,功能名3......如此类推(例如:RightClickMenuConfig="cut,copy,delete,paste,selectall,hr,quote,code,properties")

右键菜单支持的功能列表(功能名&作用):

cut 剪切
copy 复制
removeformat 清除文本格式 
delete 删除
paste 粘贴
selectall 全选
redo 重做
undo 撤销
print 打印
indent 缩进
outdent 取消缩进
hr 右键菜单增加一条分分隔线(菜单外观功能)
quote 引用样式
code 代码模式
sup 上标
sub 下标
excel  插入EXCEL表格
nowtime 插入当前时间
getpage 远程页面采集
help 在线帮助中心
link 插入超链接
search 搜索
symbol 插入特殊字符
qq 插入QQ在线图标
msn 插入MSN在线图标
icq 插入ICQ在线图标
fullscreen 全屏编辑
inserttable 插入表格
insertimage 插入图片
insertfile 插入文件
inserttemplate 插入模板
insertmedia 插入媒体文件
insertdiv 插入层
emot 插入表情
date 插入日期
calculator 辅助计算器
pastetext 粘贴文本
pasteword 粘贴WORD
properties 目标属性(根据选定的对象自动弹出相应的属性修改窗口)
resettoolbar 重置可拖曳菜单工具的位置
pagebreak 插入打印分页符

例如需要右键菜单只保留删除,复制及粘贴功能的话,只需要设置RightClickMenuConfig="delete,copy,paste"即可!
