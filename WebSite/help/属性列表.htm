DotNetTextBox控件的内置属性列表(部分属性免费版不具备)
属性值 说明
AccessKey 获取或设置使您得以快速定位到 Web 服务器控件的访问键（带下划线的字母）。
AdjustSize 设置扩展及收缩编辑框功能的增减幅度。
AutoSaveImgToLocal 是否自动保存编辑器内容里的远程图片到本地。True=是，Flase=否(默认否)
BackColor 控件编辑栏背景色。
BorderColor 控件边框样式的颜色。
BorderStyle 控件边框样式。
BorderWidth 控件边框样式的宽度。
Child 是否子控件，同一页面放置多个控件时使用。
CodeBackColor 代码样式背景色。
CodeBorder 代码样式边框粗细。
CodeBorderColor 代码样式边框色。
CodeBorderStyle 代码样式边框样式。
CodeTitle 代码样式标题文字。
EditHeight 编辑框高度。
EditBorderColor 控件编辑栏边框色。
Enabled 获取或设置一个值，该值指示是否启用 Web 服务器控件。
EnableViewState 获取或设置一个值，该值指示服务器控件是否向发出请求的客户端保持自己的视图状态以及它所包含的任何子控件的视图状态。
ExpireHours 设置控件可拖曳菜单工具栏位置记忆的失效时间(单位:小时)。默认为0(马上失效)
Focus 页面加载后是否聚焦控件编辑框。
Font 用于控件中文本的字体。(只读，带子属性)
ForeColor 控件默认文本颜色。
FrameShadow 控件编辑栏框架阴影。
GetImagesUrl 获取提交内容里所有图片的URL地址(网络路径)。例如:webeditor1.GetImagesUrl[0].ToString()(其中[i]为图片的出现顺序
GetFlashUrl 获取提交内容里所有flash的URL地址(网络路径)。例如:webeditor1.GetFlashUrl[0].ToString()(其中[i]为SWF的出现顺序
GetFlvUrl 获取提交内容里所有flv的URL地址(网络路径)。例如:webeditor1.GetFlvUrl[0].ToString()(其中[i]为FLV的出现顺序
getAutoPage 获取自动分页内容的数组集合,通过getAutoPage[i].ToString()形式获取对应分页内容。
getManualPage 获取手动分页内容的数组集合,通过getManualPage[i].ToString()形式获取对应分页内容。
getImagesPathID 设置插入图片后将图片路径即时加入页面控件(如TextBox或Input)的ID名称,默认为不插入。
Height 获取或设置 Web 服务器控件的高度。
ID 获取或设置分配给服务器控件的编程标识符。
Languages 设置界面语言。
Length 获取编辑控件内容的字符数，配合验证控件即可实现限制控件字符数。
Xhtml 转换内容为符合XHTML1.0编码规范的方式。None=不转换(默认),Client=客户端转换,Server=服务端转换
MenuConfig 设置菜单功能配置文件。文件放置在system_dntb/menuconfig/目录下
NewLineMode 回车换行的模式,默认为BR。
PageLength 设置或获取自动分页功能中每页内容的字符长度，超出此长度则自动分页。默认为2000字符。
PathType 插入文件及图片的路径形式。Relative=相对路径,AbsoluteRoot=绝对根路径,AbsoluteFull=绝对全路径"
QuoteBackColor 引用样式背景色。
QuoteBorder 引用样式边框粗细。
QuoteBorderColor 引用样式边框色。
QuoteBorderStyle 引用样式边框样式。
QuoteTitle 引样式标题文字。
RightClickMenuConfig 配置编辑器个性化右键菜单功能。(商业版功能)
Skin 设置控件皮肤文件的路径。请使用相对路径，相对于system_dntb目录，路径最后需带斜杠。
StatusHeight 状态栏高度。
Source 设置控件在代码状态时能否编辑，建议关闭。
Style 获取将在 Web 服务器控件的外部标记上呈现为样式属性的文本属性的集合。
SideMenuWidth 右侧菜单栏区域宽度。(设置为0px即可隐藏右侧菜单栏)
systemFolder 设置System目录所在的路径，请使用相对路径，路径最后需带斜杠。
TabIndex 获取或设置 Web 服务器控件的选项卡索引。
Text 控件文本值。
ToolBarBgImg 控件菜单栏背景图片的网络路径,默认为none。
ToolBarColor 控件菜单栏背景色。
ToolBarHeight 菜单栏高度。
ToolBarWidth 菜单栏宽度。
ToolTip 将鼠标放在控件时显示的工具提示。
TopMenuWidth 顶端菜单栏区域宽度。
TopMenuAlign 顶端菜单栏区域对齐方式。
TopMenuHeight 顶端菜单栏区域高度。
UploadConfig 设置上传功能配置文件的名称。
UploadFolder 设置上传文件的目录路径。路径最后需带斜杠并且相对于system文件夹。
UploadFolderSize 设置上传文件夹的空间大小。如果为0,则由上传配置文件决定空间大小。
Visible 获取或设置一个值，该值指示服务器控件是否作为 UI 呈现在页上。
Width 获取或设置 Web 服务器控件的宽度。
